Project Design


Project Overview:

  • Purpose: The purpose of this application is to provide information about each branch of the U.S military including the Army, Navy, Coast Guard, Marine Corps, Air Force, and Space Force. It will also provide functionality to see if you meet the minimum requirements of the desired branch.
  • Users: The main users of this application are people who are interested in joining one of the 6 branches of the military. It could also be used for people who just want to learn more about each branch, or news about what is going on around the world in each branch.
  • Contents: The contents of this website will be 6 main html pages, one for each military branch. Within these pages there will be an about, requirements, and news section. In each section there will be a dynamic element using javascript.

Wireframe:

Site Map:

All Pages (Will be structured the same):

  1. Name: army.html
  2. Purpose: Display information regarding the military branch including general info, news, careers, and requirements.
  3. Audience: Anyone wanting to join the specific military branch, look at news regarding the military branch, and just learning more about our military.
  4. Information: Users will be able to enter information if interested, and there will be a section to test if the user is qualified and the annual salary/benefits.
  5. Validations: There will be validations to make sure the user enters the correct information into the text fields.
  6. Contains: There will be buttons and hyperlinks
  7. Actions: When the user clicks on the page buttons, it will compute information and dynamically change what is displayed on the page

Dynamic Functionality:

  • When diffent branches of military are selected the website will change to the colors of that branch.
  • On each page there will be a requirements calculator to see if you are fit for that specific branch.
  • There will be a assessment to see which careers in the military would be best for you, and dynamically search for the results.
